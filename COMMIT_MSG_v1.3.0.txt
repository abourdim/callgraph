SUMMARY
  +399 lines since v1.2.0 (4,188 total in index.html, 502 in server.py)

── BUG FIXES ─────────────────────────────────────────

[FIX 1] Stack depth exponential DFS → memoized DP
  - Previous longestPath() was O(2^n) on dense graphs, hung on 500+ fns
  - New: memoized DFS with cycle-breaking via onStack set
  - Each node computed once, cached in _memo: O(V+E) total
  - Handles cyclic graphs gracefully (breaks at cycle, notes lower bound)

── NEW FEATURES ──────────────────────────────────────

[FEAT 1] Auto-trace main on startup
  - After indexing, auto-traces main() (or first is_entry node) at depth 3
  - Only activates when graph has >20 nodes (small projects show full graph)
  - Toast notification: "Traced from main — clear to see full graph"
  - Gives meaningful first view instead of "graph too large" error

[FEAT 2] Trace direction toggle (Both / ↓ Down / ↑ Up)
  - 3-state pill control in sidebar below depth slider
  - "Down": shows only callees (downstream tree) — no callers block
  - "Up": shows only callers (upstream tree) — no callees block
  - "Both": original behavior (callers + callees)
  - Applies to buildVisibleGraph() BFS; instant re-render

[FEAT 3] Module-aware trace nodes
  - Nodes color-coded by module (18-color palette)
  - Module name shown as second line on each node (dimmed, smaller font)
  - Module color legend appears bottom-left of canvas when >1 module visible
  - Node height increased 36→46px to accommodate module label
  - Trace role colors (caller=red, callee=green, focused=white) still override
  - External nodes retain neutral styling

[FEAT 4] Diff mode (⚖)
  - New analysis button: ⚖ Diff
  - Save current graph as baseline snapshot (localStorage)
  - Re-index after changes, then diff: shows added/removed/changed functions
  - Changed = same function name but different callee edge set
  - Added/removed shown with +/- prefix, color-coded green/red/amber
  - "Highlight in graph" button overlays diff markers on visible graph
  - "Update baseline" button to set new snapshot
  - Edge diff detail: shows +new_callee / -lost_callee per changed function

[FEAT 5] Source annotation view (📄)
  - New /api/source endpoint reads .c files with path traversal protection
  - Accessible from: info panel (📄 button), context menu ("View source")
  - Slide-in panel (right side, 50% width) shows full file with:
    · Line numbers in gutter
    · Function definition lines highlighted
    · Inline badges: callers↓ calls→ count, ⚡ISR, ☠dead, ⚠race, ◈periph
    · Click any function line → trace that function
  - Auto-scrolls to first function definition on open

[FEAT 6] Persistent bookmarks (★)
  - Star any function from info panel or right-click context menu
  - Stored in localStorage (survives session expiry, separate from graph cache)
  - ★ button in graph toolbar with badge count
  - Bookmarks panel: sorted list, module label, click to trace, click ★ to remove
  - Export bookmarks as JSON

── INTERNAL ──────────────────────────────────────────

- anaSetActive() now covers all 7 analysis buttons (was missing cycle/stack/diff)
- clearCanvas() clears module legend
- server.py: /api/source endpoint with security validation (path must be under root)
- Node height constant (NH) updated from 36→46 across all renderers
